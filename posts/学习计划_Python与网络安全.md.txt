# Python 基础 & 网络安全 学习计划

> 适用对象：有实战经验、AI辅助写代码、每天学习30-60分钟
> 预计周期：6-8个月

---

## 第一阶段：Python 基础巩固（第1-2个月）

### 目标
不靠 AI 能独立写简单脚本（发请求、处理JSON、读写文件）

### 每周安排

| 周次 | 学习内容 | 练习任务 |
|------|---------|---------|
| 第1周 | 变量、数据类型、字符串操作 | 写一个脚本：输入SKU编号，输出格式化的商品信息 |
| 第2周 | 列表、字典、嵌套数据结构 | 把一个JSON订单数据解析出来，提取关键字段 |
| 第3周 | 循环、条件判断、函数定义 | 写一个函数：传入订单列表，筛选出缺货的订单 |
| 第4周 | 文件读写、CSV/JSON处理 | 读取一个CSV文件，统计每个SKU的销量并输出 |
| 第5周 | requests 库、发送HTTP请求 | 用 requests 调用一个公开API（如天气API），打印结果 |
| 第6周 | try/except、日志打印 | 给之前的脚本加上错误处理和日志输出 |
| 第7-8周 | 综合练习 | 重写你之前做过的一个小项目（如缺货报表），不靠AI |

### 学习资源（全部免费）

- **廖雪峰 Python 教程**（文字，适合碎片时间看）
  https://www.liaoxuefeng.com/wiki/1016959663602400
- **菜鸟教程 Python3**（可以在线写代码练习）
  https://www.runoob.com/python3/python3-tutorial.html
- **B站：Python 全栈开发教程**（视频，案例多）
  https://www.bilibili.com/video/BV1wD4y1o7AS
- **Python 100天**（GitHub，每天一个小练习）
  https://github.com/jackfrued/Python-100-Days

### 学习方法
- 每天看 20 分钟教程 + 写 20 分钟代码
- 遇到工作中的需求，先尝试自己写，卡住 15 分钟再问 AI
- 每周末花 30 分钟回顾本周学的内容，写个小总结

---

## 第二阶段：HTTP 协议 & 抓包（第3-4个月）

### 目标
能独立抓包分析一个网站的 API 请求，理解完整的请求-响应流程

### 每周安排

| 周次 | 学习内容 | 练习任务 |
|------|---------|---------|
| 第1周 | HTTP 协议基础（方法、状态码、Headers） | 用浏览器 F12 分析 5 个不同网站的请求 |
| 第2周 | Cookie、Session、Token 认证机制 | 分析一个需要登录的网站，找出登录流程 |
| 第3周 | Fiddler/Charles 安装使用 | 抓手机APP的包，分析请求格式 |
| 第4周 | Wireshark 基础（TCP三次握手、DNS） | 抓一次完整的网页加载流量，分析每个包 |
| 第5周 | HTTPS 与 SSL/TLS 原理 | 理解为什么 HTTPS 抓包需要装证书 |
| 第6周 | mitmproxy 使用（Python中间人代理） | 写一个 mitmproxy 脚本，自动修改某个请求 |
| 第7-8周 | 综合实战 | 分析 Shopee 或 Mabang 的某个 API 完整流程 |

### 学习资源

- **HTTP 协议入门**（阮一峰，图文并茂）
  https://www.ruanyifeng.com/blog/2016/08/http.html
- **Wireshark 入门教程**（B站搜"Wireshark 入门"）
- **mitmproxy 官方文档**
  https://docs.mitmproxy.org/stable/
- **Fiddler 中文教程**（B站搜"Fiddler 抓包教程"）

### 学习方法
- 结合工作实战：每次调试 Mabang/Shopee 接口时用 Wireshark 同步抓包
- 每天抓一个网站的包，写下分析笔记
- 重点理解：请求头每个字段的含义、Cookie 如何传递、API 签名如何生成

## 第三阶段：加密基础（第5个月）

### 目标
理解常见加密算法原理，能看懂代码中的加密逻辑

### 学习内容

| 周次 | 学习内容 | 练习任务 |
|------|---------|---------|
| 第1周 | 编码 vs 加密 vs 哈希的区别 | 用 Python 实现 Base64 编解码 |
| 第2周 | MD5、SHA256 哈希算法 | 用 hashlib 计算文件的 MD5 值 |
| 第3周 | HMAC 签名（结合你做过的 Mabang API） | 手写一个 HMAC-MD5 签名函数，对比之前AI写的 |
| 第4周 | AES 对称加密、RSA 非对称加密 | 用 PyCryptodome 实现 AES 加解密 |

### 学习资源

- **密码学入门**（阮一峰）
  https://www.ruanyifeng.com/blog/2006/12/notes_on_erta.html
- **Python hashlib 官方文档**
  https://docs.python.org/3/library/hashlib.html
- **B站搜"密码学 通俗讲解"**（看播放量高的）

## 第四阶段：网络安全入门（第6-8个月）

### 目标
了解常见攻防手法，能在靶场环境独立完成基础题目

### 学习内容

| 周次 | 学习内容 | 练习任务 |
|------|---------|---------|
| 第1-2周 | Web 安全基础（SQL注入、XSS、CSRF） | TryHackMe 基础房间 |
| 第3-4周 | 信息收集与端口扫描（Nmap） | 在靶机上做一次完整的信息收集 |
| 第5-6周 | 流量分析与取证（Wireshark进阶） | 分析 CTF 题目中的 pcap 文件 |
| 第7-8周 | 漏洞利用基础 | Hack The Box 简单靶机 |

### 学习资源（全部免费）

- **TryHackMe**（新手最友好的靶场，有免费房间）
  https://tryhackme.com
  推荐路径：Complete Beginner → Web Fundamentals → Network Security
- **Hack The Box**（进阶靶场）
  https://www.hackthebox.com
- **CTFHub**（中文 CTF 练习平台）
  https://www.ctfhub.com
- **B站搜"网络安全入门 零基础"**

## 日常习惯建议

1. **工作日**：每天 30 分钟，看教程或做练习
2. **周末**：每周抽 1-2 小时做综合练习或靶场
3. **工作中**：遇到 API 对接，多花 10 分钟自己理解代码再继续
4. **记笔记**：用 Markdown 记录学习笔记，存到你的 GitHub 仓库
5. **不要贪多**：每天学一个小知识点就够了，坚持比速度重要
